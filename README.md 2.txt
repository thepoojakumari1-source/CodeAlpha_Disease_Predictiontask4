# Disease Prediction using Machine Learning

## Objective
To predict whether a patient has diabetes based on medical data.

## Dataset
Pima Indians Diabetes Dataset (Kaggle)

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## Model Used
- Logistic Regression

## Steps Performed
- Data loading
- Data cleaning
- Train-test split
- Model training
- Model evaluation

## Result
Achieved good accuracy on test data.
